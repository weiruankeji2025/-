# 威软吃瓜视频助手

> cgtt.me（吃瓜网）视频自动下载油猴脚本

---

## 功能特性

- **自动抓取视频** — 通过扫描 `<video>` 标签、页面脚本内容、以及 Hook XHR/fetch 网络请求，全面捕获页面中的视频资源（MP4 / M3U8 / FLV）
- **最高质量优先** — 对同一视频的多个清晰度源自动排序，优先选取最高画质（4K > 1080P > 720P …）
- **用户自主选择** — 弹出独立下载面板，每条资源均提供独立下载按钮，用户决定是否下载
- **视频封面下载** — 自动从 Open Graph / Twitter Card / JSON-LD 等元数据中提取封面，一键下载封面图
- **一键全部下载** — 同时下载视频文件与封面图
- **动态资源监听** — MutationObserver + 网络 Hook 实时监听，播放视频后自动识别新资源
- **无感融合** — 固定悬浮按钮，不干扰正常浏览体验

## 安装方法

### 前提

安装任意油猴扩展：

| 扩展 | 浏览器 |
|------|--------|
| [Tampermonkey](https://www.tampermonkey.net/) | Chrome / Edge / Firefox / Safari |
| [Violentmonkey](https://violentmonkey.github.io/) | Chrome / Firefox |

### 步骤

1. 打开油猴扩展 → 新建脚本
2. 将 `cgtt-video-downloader.user.js` 的全部内容粘贴进去
3. 保存（Ctrl+S）
4. 访问 `cgtt.me` 即可看到右下角的 **🎬 视频下载** 按钮

---

## 使用说明

1. 进入 cgtt.me 任意视频页面
2. 点击右下角 **🎬 视频下载** 浮动按钮，打开下载面板
3. 脚本自动扫描页面中的视频资源
4. 对每条资源可选择：
   - **⬇ 下载视频** — 仅下载视频文件
   - **⬇ 封面** — 仅下载封面图片
   - **⬇ 全部** — 同时下载视频与封面
5. 若视频为动态加载（点击播放后才出现），点击面板中的 **重新扫描** 按钮即可
